Ajax

Ajax (에이잭스, 아작스) : 화면 깜박임 없이 화면의 일부를 업데이트(=갱신) 하는 기술

데이터 요청 방식(HTTP 프로토콜)

  • post : Create
  • get : Read
  • put : Update
  • delete: Delete

CRUD라고 해서, 데이터를 생성하고 요청(=읽고), 갱신(=업데이트), 삭제하는 작업을 하게 됩니다.

다만, HTML5에서는 GET, POST만 Form을 통해서 요청할 수 있고 PUT, DELTE는 AJAX로 요청해야 함. 기술적으로, 그렇게 만들어져 있음(=원 저작자가 어떠한 의도로 그렇게 설계했다고 알려져 있음)

form을 통해 GET, POST 요청하기(예시)

id, pw 입력하고 name, email 입력받아서 서버로 전송하면 계정이 생성되는 것
