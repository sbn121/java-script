 :: 자바스크립트/jQuery 문장 구조

1) 대,소문자 구분
2) 문장의 끝에 세미콜론(;) 구분
3) 주석은 한줄 주석(//) 과 여러줄 주석(/*  ... */)을 사용
4) 자료형 : 숫자, 문자열, 배열, 객체, 함수, 심볼, null, undefined, class [ES6 이상] 등이 있음
5) 식별자 규칙 : 영문자, _, $을 사용하고 숫자로 시작할 수 없음.
6) 자바스크립트는 자료형이 (데이터에 따라서) 동적할당됨 ==> TypeScript [타입스크립트]
7) 변수 선언 키워드

var : ES5 버전에서 (과거/오랫동안 사용~ 지금도, 하지만 가급적이면 앞으로는??) / 변수
※ 선언이나 할당/재할당에서 발생할수 있는 문제가 있어서 var는 가급적 피하자!

let :  ES6 버전 (지금부터 앞으로 오랫동안 사용할...가급적 사용)   /   변수 
const : ES6 버전  " (상동)   /  상수
